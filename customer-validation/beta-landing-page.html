C2PA (Coalition for Content Provenance and Authenticity) is an open standard for content provenance. CredLink implements C2PA to create cryptographic proofs that show who created content, when it was created, and whether it has been altered.

When content is signed, we create a cryptographic proof that includes metadata about the content, creator, and timestamp. When content is verified, we check that the content matches the original proof and hasn't been tampered with.
