Introduction

The CredLink API provides programmatic access to sign and verify digital images with C2PA content credentials. Our API enables developers to integrate content authenticity directly into their applications.

Base URL: https://api.credlink.com
Current Version: v1.0.0
Status: ‚óè Live

  • ‚úÖ Sign JPEG, PNG, and WebP images with C2PA manifests
  • ‚úÖ Verify content authenticity and provenance
  • ‚úÖ Role-based access control (Admin, User, Readonly)
  • ‚úÖ Enterprise-grade security (100/100 security score)
  • ‚úÖ Real-time verification with confidence scores
  • ‚úÖ Comprehensive audit trails and logging

Authentication

All API requests must be authenticated using an API key sent in the X-API-Key header. 

curl -H "X-API-Key: your-api-key" https://api.credlink.com/health

API Key Tiers

Tier Rate Limit Features
Free 10 requests/minute Basic signing and verification
Pro 100 requests/minute Advanced features, priority support
Enterprise 1000 requests/minute Full access, custom integrations

Rate Limits

API requests are rate limited based on your API key tier. Rate limit headers are included in every response: 

X-RateLimit-Limit: 100
X-RateLimit-Remaining: 95
X-RateLimit-Reset: 1640995200
Note: Exceeding rate limits will result in HTTP 429 status code.

Error Codes

The API uses standard HTTP status codes and includes detailed error information.

Status Code Meaning Description
200 OK Request successful
400 Bad Request Invalid request parameters
401 Unauthorized Missing or invalid API key
403 Forbidden Insufficient permissions
404 Not Found Resource not found
429 Too Many Requests Rate limit exceeded
500 Internal Server Error Server error occurred

Code Examples

JavaScript/Node.js

const axios = require('axios');

// Sign an image
async function signImage(imageData, creator) {
  try {
    const response = await axios.post('https://api.credlink.com/api/sign', {
      image: imageData,
      creator: creator,
      title: 'My Authentic Image'
    }, {
      headers: {
        'X-API-Key': 'your-api-key',
        'Content-Type': 'application/json'
      }
    });
    
    return response.data;
  } catch (error) {
    console.error('Signing failed:', error.response.data);
  }
}

// Verify an image
async function verifyImage(manifestId) {
  try {
    const response = await axios.get(`https://api.credlink.com/api/verify/${manifestId}`);
    return response.data;
  } catch (error) {
    console.error('Verification failed:', error.response.data);
  }
}

Python

import requests
import base64

# Sign an image
def sign_image(image_path, creator, api_key):
    with open(image_path, 'rb') as f:
        image_data = base64.b64encode(f.read()).decode()
    
    response = requests.post('https://api.credlink.com/api/sign', 
        json={
            'image': image_data,
            'creator': creator,
            'title': 'My Authentic Image'
        },
        headers={
            'X-API-Key': api_key,
            'Content-Type': 'application/json'
        }
    )
    
    return response.json()

# Verify an image
def verify_image(manifest_id):
    response = requests.get(f'https://api.credlink.com/api/verify/{manifest_id}')
    return response.json()
