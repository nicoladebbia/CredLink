# C2PA Concierge - Vulnerability Disclosure Program

## Program Overview
ISO/IEC 29147 & 30111 compliant VDP for responsible security research

## Scope
**In-Scope:**
- c2pa-concierge.org main application
- API endpoints (api.c2pa-concierge.org)
- Cloudflare Workers
- Billing systems (Stripe integration)
- Verify endpoints and manifest processing
- Badge UI components

**Out-of-Scope:**
- Third-party services
- Physical infrastructure
- Social engineering
- DoS attacks

## Safe Harbor
Legal protection for good-faith security research following:
- Authorized testing only
- No data exfiltration
- Responsible disclosure
- Compliance with applicable laws

## Reporting
**Contact:** security@c2pa-concierge.org
**PGP Key:** Available on security page
**Response SLA:**
- Acknowledgment: ≤72 hours
- Triage: ≤7 days
- Fix timeline: Based on severity

## Severity Classification
- **Critical:** Remote code execution, data breach
- **High:** Privilege escalation, financial impact
- **Medium:** Limited impact, user data exposure
- **Low:** Information disclosure, minor issues

## Rewards
Recognition program (no monetary rewards):
- Hall of Fame listing
- Security researcher badge
- Conference speaking opportunities
- Exclusive swag and recognition

## Coordination
- Private disclosure before public
- 90-day disclosure timeline
- Vendor coordination for third-party issues
- CVE assignment assistance

## Legal
Compliance with:
- ISO/IEC 29147 (Vulnerability disclosure)
- ISO/IEC 30111 (Vulnerability handling)
- NIST Cybersecurity Framework
- GDPR data protection
