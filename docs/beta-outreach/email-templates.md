## Personalization Guide

### Required Personalization (Every Email)

1. **[Name]** - First name only (John, not John Smith)
2. **[Company]** - Company name
3. **[specific work]** - What they're doing (e.g., "building trust features")
4. **[specific benefit]** - How CredLink helps them (e.g., "verify user photos")

### Optional Personalization (High-Priority Prospects)

5. **Recent news** - Funding, product launch, hiring
6. **Specific pain point** - Known problem they're solving
7. **Competitor mention** - "Similar to what [Competitor] is doing"
8. **Mutual connection** - "I saw you worked with [Person]"
9. **Content reference** - "Read your blog post about [Topic]"

### Research Sources

- LinkedIn profile (recent posts, about section)
- Company blog (recent articles)
- TechCrunch (funding announcements)
- Twitter/X (recent tweets)
- Company careers page (what they're hiring for)

### Personalization Examples

**Generic:**
> Your team at Shopify is doing great work and might benefit from image verification.

**Personalized:**
> I saw Shopify recently launched seller verification features. CredLink could help verify product photos are authentic, reducing "not as described" returns.

**Generic:**
> Your team at New York Times handles news photos.

**Personalized:**
> I noticed NYT's recent fact-checking initiative around manipulated images. CredLink could help verify photo authenticity in breaking news.

## Subject Lines

### High-Performing Subject Lines

**Direct:**
- Early access to image authenticity API (beta)
- [Company] + image authenticity
- Quick question about [Company]'s image verification

**Problem-Focused:**
- Solving [specific problem] with cryptographic proofs
- How [Company] can verify image authenticity
- Reduce [specific problem] with C2PA proofs

**Value-First:**
- Free beta: Image authenticity API
- Verify images after 1,000+ shares
- Combat [specific problem] with cryptographic proofs

**Personalized:**
- Following up on [Company]'s [recent news]
- Saw your post about [topic]
- [Mutual connection] suggested I reach out

### Subject Line Best Practices

✅ **Do:**
- Keep under 50 characters
- Use their company name
- Be specific about value
- Ask a question
- Reference recent news

❌ **Don't:**
- Use all caps
- Use excessive punctuation!!!
- Be vague or generic
- Use "Re:" if not a reply
- Use misleading subjects

## Best Practices

### Email Structure

**Opening (1-2 sentences):**
- Hook with problem or value
- Show you researched them

**Body (2-3 sentences):**
- Explain solution briefly
- Show how it helps them specifically
- Provide social proof (optional)

**Call-to-Action (1 sentence):**
- Clear next step
- Link to apply
- Low friction

**Closing (1 sentence):**
- Offer to answer questions
- Keep it casual

### Writing Tips

✅ **Do:**
- Keep it under 150 words
- Use short sentences
- Break into paragraphs
- Use bullet points
- Make it scannable
- Be conversational
- Show personality

❌ **Don't:**
- Write long paragraphs
- Use jargon excessively
- Be too formal
- Oversell
- Use attachments
- Include multiple CTAs

### Timing

**Best Days to Send:**
- Tuesday: 8am-10am
- Wednesday: 8am-10am
- Thursday: 8am-10am

**Avoid:**
- Monday (inbox overload)
- Friday (checking out)
- Weekends (unprofessional)
- Holidays

**Follow-Up Schedule:**
- Day 0: Initial email
- Day 3: Follow-up #1
- Day 7: Follow-up #2
- Day 14: Final follow-up (optional)

### Response Handling

**If interested:**
- Respond within 1 hour
- Send application link
- Offer to answer questions
- Schedule call if needed

**If not interested:**
- Thank them for responding
- Ask if timing might be better later
- Request referral to someone else
- Add to nurture list

**If no response:**
- Follow up 2-3 times
- Then move to nurture list
- Try different channel (LinkedIn)

## A/B Testing

### Test Variables

1. **Subject line** - Direct vs. question
2. **Length** - Short (100 words) vs. long (200 words)
3. **CTA** - "Apply now" vs. "Learn more"
4. **Social proof** - With vs. without
5. **Personalization** - High vs. low

### Testing Process

1. Split prospects into 2 groups
2. Send version A to group 1
3. Send version B to group 2
4. Track metrics for 3-5 days
5. Use winner for remaining prospects

## Legal Compliance

### CAN-SPAM Requirements

✅ Include:
- Real "From" name and email
- Accurate subject line
- Physical address
- Unsubscribe link (if bulk)

❌ Don't:
- Use deceptive subject lines
- Hide sender identity
- Ignore unsubscribe requests

### GDPR Compliance

✅ Do:
- Have legitimate interest
- Provide opt-out
- Respect privacy
- Delete on request

❌ Don't:
- Buy email lists
- Spam
- Ignore opt-outs

- 5+ responses (15% response rate)
- 2+ applications (5% application rate)

### Optimization

If open rate < 40%:
- Test new subject lines
- Verify email deliverability
- Check spam score

If response rate < 5%:
- Improve personalization
- Shorten email
- Clarify value proposition

If application rate < 2%:
- Simplify application process
- Add more social proof
- Offer call instead
