# Email Finding Tools & Techniques

Comprehensive guide to finding and verifying decision maker emails.

---

## Free Methods

### 1. Email Pattern Guessing

**Common Patterns:**
```
firstname@company.com
firstname.lastname@company.com
firstnamelastname@company.com
f.lastname@company.com
flastname@company.com
first.last@company.com
```

**How to Find Pattern:**
1. Find ANY employee email from company
2. Identify the pattern
3. Apply to your target contact
4. Verify before sending

**Sources for Sample Emails:**
- Company website (contact page, team bios)
- Press releases
- GitHub commits
- Blog post authors
- Support emails

---

### 2. Google Search Techniques

**Basic Search:**
```
"John Smith" "Company Name" email
"John Smith" "@company.com"
```

**Advanced Search:**
```
site:company.com "John Smith" email
site:github.com "John Smith" "Company Name"
site:twitter.com "John Smith" "Company Name" email
intext:"John Smith" intext:"@company.com"
```

**Find Email Pattern:**
```
site:company.com "@company.com"
"@company.com" -site:company.com
```

## Advanced Techniques

### 1. Reverse Email Search

Find email, then identify person:
```
site:linkedin.com "email@company.com"
"email@company.com" -site:company.com
```

### 2. Email Permutator

Generate all possible combinations:
- firstname.lastname@company.com
- firstnamelastname@company.com
- f.lastname@company.com
- etc.

Then verify each one.

### 3. WHOIS Lookup

For smaller companies:
```
whois company.com
```

Shows registrant email (sometimes decision maker).

### 4. Press Release Search

```
site:prnewswire.com "Company Name" "contact"
site:businesswire.com "Company Name" "email"
```

Press contacts often include emails.

### 5. Conference Speaker Lists

Speakers often list emails:
- Conference websites
- Speaker bios
- Presentation slides

## Legal & Ethical Considerations

### GDPR (Europe)
- Need legitimate interest
- Allow opt-out
- Don't spam
- Respect privacy

### CAN-SPAM (US)
- Include unsubscribe link
- Use real sender address
- Don't use misleading subject
- Honor opt-outs quickly

### Best Practice
- Only email business contacts
- Use work emails only
- Provide value in email
- Allow easy opt-out
- Respect "no" responses

## Troubleshooting

### "Email not found"
- Try different name variations
- Check LinkedIn for middle name
- Try company website
- Use contact form

### "Low confidence score"
- Verify with multiple tools
- Try alternate patterns
- Look for social proof
- Consider skipping

### "Email bounces"
- Person may have left company
- Email pattern may be wrong
- Domain may have changed
- Remove from list

### "No response"
- Email may be wrong
- May not check email often
- Try LinkedIn instead
- Follow up once
