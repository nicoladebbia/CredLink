# Extension Icons

This directory contains the icons for the C2 Concierge browser extension.

## Required Icons:
- icon16.png - 16x16 pixels for favicon
- icon48.png - 48x48 pixels for extension management
- icon128.png - 128x128 pixels for Chrome Web Store

## Design Guidelines:
- Use the C2 Concierge brand colors
- Include a verification/checkmark symbol
- Ensure visibility on both light and dark backgrounds
- Follow Chrome Web Store icon guidelines
